# 智慧型股票投資人

## Chap 1.投資與投機
* 投資操作必須以深入分析(公司及業務是否穩定)為基礎，確保本金的安全(避免重大損失），並獲得適當的報酬(不要期望過高)
* 價格無法預測

## Chap 2. 投資者與通貨膨脹
* 通貨膨脹無法避免
* 如何抵抗通脹
  * REITs (不動產信用委託)
  * TIPS (抗通貨膨脹保值債卷)

## Chap 3. 
* 本益比
* 股息
* 資本獲利率
1. 不要借錢買/持有股票
2. 🙅增加持有股票的資金比重
3. 絕對不能只根據過去的數據來預測未來
4. 減低期望值
5. 公司實際成長 (revenue, dividend increase, 1.5-2%) + inflation rate (2.4%) + 投資情緒的加減 (1.9%) = 報酬
